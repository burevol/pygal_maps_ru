# pygal_maps_ru  

Russian maps for Pygal

##Usage

~~~~
from pygal.maps.ru import Regions, Departments

map = Regions()
map.add('Data', {'11': 2, '22': 5})
map.render()
~~~~

##Requirements

* [Pygal](https://github.com/Kozea/pygal)
